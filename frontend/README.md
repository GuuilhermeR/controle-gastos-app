# Frontend - Controle de Gastos Pessoal

Interface React com Ant Design para registrar entradas/despesas e acompanhar dashboards financeiros.

## Rodar localmente

```bash
npm install
npm run dev
```

Por padrão, o Vite abre em `http://localhost:5173` e redireciona `/api` para `http://localhost:5000`.

## Build

```bash
npm run build
```

## Variáveis

- `VITE_API_URL`: URL pública da API quando o frontend estiver hospedado separado do backend.
